
# El Codi
Un joc diari per descobrir la combinació correcta de botons i obrir la caixa forta.

## Com Jugar
- Clica els botons en l'ordre correcte per descobrir la combinació diària.
- Si encertes en pocs intents, guanyaràs més punts XP.

## Tecnologies Utilitzades
- HTML, CSS, JavaScript
- Llibreria SeedRandom per a la generació de seqüències.

## Contribució
- Fes un fork d'aquest repositori, crea una nova branca, fes els teus canvis i envia un pull request.
